--- layout: post title: HTML5 Meetup October Recap date: '2010-10-19' categories: - Mobile Web tags: [] status: publish type: post published: true meta: _edit_last: '1' comments: true --- Mostly out of completeness reasons here is a little recap of our October 7th HTML5 Meetup:

This time 3 out of the 7 total attendees were there for the first time, so we might actually grow this meetup one day. But it was good as it was and we had some quite interesting topics. I just want to mention them here quickly (as far as I can remember):
  • Book: HTML5 for Web Designers
  • HbbTV and why it might be interesting for web developers
  • Google Web Fonts (thanks to Sven!)
  • Amazon Kindle product presentation ;)
  • Zepto.js - a minimalist JavaScript framework for mobile WebKit browsers
  • Jo - HTML5 Mobile App Framework
  • vapor.js - very fast and fun JavaScript tool
  • Unify - JavaScript mobile application framework
  • CSS3 vs. Canvas animations (CSS3 wins)
  • Dextrose AG acquisition by Zynga (both win, they're hiring)
  • (function(){}()) vs. (function(){})() (thanks Björn)
It was a fun evening at Cafe Brücke (their food is much better than their website). It was surprisingly full between 20:00 to 22:00, but quieter than our previous locations.

Thank you all for coming and see you next time!
